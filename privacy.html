Données collectées

Ronin Traq collecte uniquement les données nécessaires au fonctionnement de l'application :

  • Données d'identification : adresse email, pseudonyme choisi
  • Localisation GPS lors de l'enregistrement d'activités sportives
  • Données d'activité : distance, dénivelé, durée, tracé GPS
  • Données de connexion : date, heure, type d'appareil

Nous ne collectons pas de données de santé médicales, de données bancaires, ni d'informations sensibles au sens du RGPD. Aucun tracking publicitaire. Aucun cookie.

Finalités et bases légales

  • Créer et gérer votre compte (exécution du contrat)
  • Enregistrer vos activités sportives (exécution du contrat)
  • Partager des parcours avec la communauté (consentement explicite)
  • Améliorer les fonctionnalités de l'application (intérêt légitime)

Localisation GPS

L'accès à votre position GPS est utilisé exclusivement lors de l'enregistrement d'une activité que vous déclenchez vous-même. Aucune collecte en dehors de ces moments.

Cette permission peut être révoquée à tout moment dans les paramètres de votre téléphone (iOS : Réglages → Confidentialité → Service de localisation ; Android : Paramètres → Applications → Ronin Traq → Autorisations).

Partage des données

Vos données ne sont jamais vendues ni louées à des tiers. Elles peuvent être transmises uniquement aux prestataires techniques suivants :

  • Supabase — hébergement base de données (serveurs en Europe)
  • Apple / Google — gestion des abonnements via les stores
  • RevenueCat — gestion des achats in-app
  • Autres utilisateurs — uniquement pour les contenus que vous choisissez de rendre publics

Durée de conservation

  • Données de compte et d'activité : conservées pendant toute la durée d'activité du compte
  • Suppression : toutes les données sont effacées dans les 30 jours suivant la clôture du compte
  • Données de facturation : conservées 10 ans conformément aux obligations légales

Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (UE 2016/679), vous disposez des droits suivants :

Accès Obtenir une copie de vos données
Rectification Corriger vos données inexactes
Effacement Supprimer votre compte et données
Portabilité Récupérer vos données en format lisible
Opposition Vous opposer à certains traitements
Limitation Restreindre le traitement de vos données

Pour exercer ces droits, contactez-nous à contact@ronintraq.fr. Délai de réponse : 30 jours.
Vous pouvez également déposer une réclamation auprès de la CNIL (cnil.fr).

Sécurité

  • Données chiffrées en transit (HTTPS/TLS)
  • Données chiffrées au repos sur les serveurs Supabase
  • Authentification sécurisée gérée par Supabase Auth
  • Aucun mot de passe stocké en clair

Modifications

Toute modification substantielle de cette politique sera notifiée par email ou via une notification dans l'application, au moins 7 jours avant son entrée en vigueur.
